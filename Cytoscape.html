Cytoscape

Características

Cytoscape es una plataforma de software de bioinformática de código abierto para visualizar redes de interacción molecular e integrarse con perfiles de expresión genética y otros datos estatales. Las funciones adicionales están disponibles como complementos. Los complementos están disponibles para análisis de perfiles moleculares y de red, nuevos diseños, compatibilidad con formatos de archivo adicionales y conexión con bases de datos y búsqueda en redes grandes. Cualquier persona puede desarrollar complementos utilizando la arquitectura de software Java abierta de Cytoscape y se recomienda el desarrollo de la comunidad de complementos. Cytoscape también tiene un proyecto hermano centrado en JavaScript llamado Cytoscape.js que se puede usar para analizar y visualizar gráficos en entornos JavaScript, como un navegador.

¿Qué puedo hacer con Cytoscape?

Biologia

Cytoscape admite muchos casos de uso en biología molecular y de sistemas, genómica y proteómica:

  • Cargar conjuntos de datos de interacción genética y molecular en muchos formatos estándar
  • Proyectar e integrar conjuntos de datos globales y anotaciones funcionales
  • Establezca potentes mapeos visuales a través de estos datos
  • Realice análisis y modelado avanzados con Cytoscape Apps
  • Visualize and analyze human-curated pathway datasets such as WikiPathways, Reactome, and KEGG.

    • Ciencias Sociales

  • Visualizar y analizar grandes redes sociales de relaciones interpersonales.
  • Ensamblar redes sociales a partir de tablas y formularios
  • Reúne las interacciones sociales de la web mediante una variedad de API's de servicios web con lenguajes de secuencias de comandos y guárdelas en formatos de archivo de datos estándar. Cytoscape admite la mayoría de los formatos de archivo estándar.
  • Calcula las estadísticas de la red usando aplicaciones
  • Úselo con otras herramientas, como R con el paquete sna / igraph o NetworkX, para un análisis más avanzado

    • Análisis general de redes complejas

  • Calcule estadísticas de redes por aplicaciones como NetworkAnalyzer o CentiScaPe
  • Encuentra el camino más corto
  • Encuentre clústeres mediante varios tipos de algoritmos
  • Úselo con otras herramientas para un análisis más avanzado
  • Realice análisis de red avanzados en herramientas populares, como igraph, Pajek o GraphViz, e impórtelo a Cytoscape como formatos de archivo estándar como GraphML

Desarrollo de aplicaciones

Cytoscape es expandible y extensible. Cytoscape tiene una vibrante comunidad de desarrolladores de aplicaciones y más de cien aplicaciones desarrolladas por terceros.

  • El código fuente está disponible en GitHub
  • Ejecutado en el marco OSGi
  • Desarrolle aplicaciones simples y complejas en Java
  • Interfaz de programación de aplicaciones (API) limpia y rica de Cytoscape
  • Escriba funciones simples con lenguajes de secuencias de comandos, incluidos Ruby, Python, Groovy y JavaScript
