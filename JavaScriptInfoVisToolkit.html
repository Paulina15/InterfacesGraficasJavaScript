JavaScript InfoVis Toolkit

Características

El kit de herramientas de JavaScript InfoVis proporciona herramientas para crear visualizaciones de datos interactivas para la Web. El kit de herramientas implementa características avanzadas de visualización de información como TreeMaps, una visualización adaptada de árboles basada en el SpaceTree, una técnica de enfoque + contexto para trazar árboles hiperbólicos, un diseño radial de árboles con animaciones avanzadas llamado RGraph y otras visualizaciones. En noviembre de 2010, el kit de herramientas fue adquirido por la Fundación Sencha Labs. El kit de herramientas de JavaScript InfoVis ha sido elegido como organización y proyecto de tutoría para Google Summer of Code 2011.
