JAVASCRIPT

Librerias Graficas.

Una biblioteca de gráficos JavaScript es una colección de funciones que se utilizan para ayudar en la creación de gráficos para el elemento lienzo HTML5 o SVG. Estas bibliotecas ayudan en el desarrollo y visualización de elementos gráficos como partículas, movimiento, animación, trazado y SVG, gráficos 3D y, en la mayoría de los casos, también proporcionan un acceso más fácil a eventos JavaScript. La mayoría de estas bibliotecas gráficas son API en nombre y función, pero se pueden comparar por su mérito como bibliotecas gráficas.
