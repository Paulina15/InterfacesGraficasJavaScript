RGraph

Características

RGraph es una biblioteca moderna de tablas y gráficos basada en JavaScript para sitios web modernos. Su objetivo es ser la mejor biblioteca de gráficos de código abierto que pueda tener en sus manos y con la cantidad de diferentes tipos de gráficos que han aumentado a 50-60, utilizando tanto lienzo HTML5 como SVG, RGraph tiene un gran atractivo y puede representar muchos tipos de datos de forma rápida y eficiente.

Ventajas

  • Gráficos receptivos
